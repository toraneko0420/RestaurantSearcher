## アプリケーション機能

### 機能一覧
- レストラン検索：ホットペッパーグルメサーチAPIを使用して、現在地周辺の飲食店を検索する。
- レストラン情報取得：ホットペッパーグルメサーチAPIを使用して、飲食店の詳細情報を取得する。
- 地図アプリ連携：飲食店の所在地を地図アプリに連携する。

### 画面一覧
- 検索画面 ：現在地からの検索半径を指定し、お店の種類を指定してレストランを検索
- 結果画面 ：検索結果の飲食店の名前と最寄り表示
- 詳細画面 ：検索結果の飲食店を一覧表示


### 使用しているAPI,SDK,ライブラリなど
- ホットペッパーグルメサーチAPI
